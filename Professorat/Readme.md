# Professorat

Carpeta destinada al professorat del mòdul de Desenvolupament Web en Entorn Client (DWEC). Conté tota la documentació necessària per a la planificació, avaluació i desenvolupament del curs.

---

## Estructura de carpetes

### [Avaluació](./Avaluacio/)

Documentació relativa a l'avaluació de l'alumnat. Inclou:

- [Rubricas.md](./Avaluacio/Rubricas.md) - Rúbriques d'avaluació per a les diferents tasques i projectes.
- [README.md](./Avaluacio/README.md) - Explicació detallada del sistema d'avaluació, criteris i ponderació.
- [Plantilla de qualificació.xlsx](./Avaluacio/Plantilla_qualificació.xlsx) - Document Excel per a registrar i calcular les puntuacions finals.

### [Planificació temporal](./Planificacio_temporal/)

Organització temporal del projecte, des dels Resultats d'Aprenentatge (RA) fins als continguts detallats per sprint:

- [Organitzacio_projecte.md](./Planificacio_temporal/Organitzacio_projecte.md) - Desglossament de RA, continguts per sprint i estructura del projecte.
- [Diagrama de Gantt.ods](./Planificacio_temporal/Diagrama%20de%20Gantt.ods) - Diagrama de Gantt amb la planificació temporal del curs.
- [CALENDARI ESCOLAR 2025-2026.jpg](./Planificacio_temporal/CALENDARI%20ESCOLAR%202025-2026.jpg) - Calendari escolar de referència per a la planificació.
- [infografia.png](./Planificacio_temporal/infografia.png) - Infografia resum de l'organització del projecte.

### [Materials i recursos](./materials_recursos/)

Materials i recursos didàctics per a impartir les classes. Inclou:

- **[Dossier d'activitats](./materials_recursos/Dossier_d_activitats.pdf)** - Document que conté totes les activitats per fer. Aquestes activitats estan **ordenades per sprint i ordre d'execució**. Cada activitat descrita en aquest dossier especifica la seua pròpia rúbrica i està vinculada directament a les tasques del tauler Kanban de Trello, assegurant la traçabilitat entre el treball planificat i l'avaluació.
- **[Teoria](./materials_recursos/Teoria/)** - Teoria a impartir durant tot el curs, organitzada per sprints i unitat. Cada sprint conté:
    - Presentacions de referència (PDF) per a guiar la impartició de les classes.
    - Documents amb tota la teoria a impartir, amb exemples i per unitats.
- **[Materials.md](./materials_recursos/Materials.md)** - Llistat complet de tots els materials i eines.

### [Guia didàctica](./Guia%20did%C3%A0ctica/)

Guia didàctica completa del mòdul:

- [Guia_Didactica.md](./Guia%20did%C3%A0ctica/Guia_Didactica.md) - Document principal amb la planificació didàctica.
- [Relació RA (saber fer-saber estar).ods](./Guia%20did%C3%A0ctica/Relaci%C3%B3%20RA%20(saber%20fer-saber%20esta...).ods) - Relació de Resultats d'Aprenentatge (saber fer - saber estar).

### [Metodologia]()

- [estrategies_metodologiques.md](./estrategies_metodologiques.md) - Estratègies metodològiques a seguir durant el curs.
> **Nota metodològica**: El ritme d'impartició de la teoria no està fixat per sessions, sinó que s'ha d'adaptar al ritme de cada classe. Cal prioritzar una ensenyança participativa, fent que l'alumnat active els coneixements previs i participe activament. És imprescindible deixar temps suficient tant per a la realització d'activitats com per al treball autònom en el projecte. Les activitats del dossier s'han d'anar proporcionant a l'alumnat a mesura que s'avança en cada sprint, de forma progressiva i coherent amb els continguts impartits.
---

## Com utilitzar esta carpeta

1. **Al començar el curs**: Revisa la *Guia didàctica* i la *Planificació temporal* per a entendre l'organització general.
2. **Per a preparar les classes**: Consulta la carpeta *materials_recursos/Teoria* segons el sprint que vages a impartir.
3. **Per a avaluar**: Consulta *Avaluació* per a les rúbriques i criteris d'avaluació.
4. **Per a la planificació temporal**: Usa el *Diagrama de Gantt* i el *Calendari escolar* com a referència.
