# SA Comerç Local

**Mòdul 0612 — Desenvolupament Web en Entorn Client (DWEC)**

---

## En què consistix este projecte?

**SAComercLocal** és un projecte que simula el desenvolupament complet d'una aplicació web per a una plataforma de comerços locals. L'alumnat, organitzat en equips, haurà de dissenyar i construir una web que permeta als usuaris navegar per un catàleg de comerços locals, consultar les seues fitxes, buscar i filtrar per categories, contactar amb els establiments i veure la seua ubicació en un mapa interactiu.

El projecte està dissenyat per a treballar de manera transversal tots els continguts del mòdul 0612 al llarg de **5 sprints**, aplicant la metodologia **Scrum** i la gestió de projecte mitjançant un **tauler Kanban (Trello)**.

### Objectius principals

- Comprendre el funcionament de les aplicacions web en entorn client.
- Dominar JavaScript modern i el framework React.
- Aplicar la metodologia Scrum amb treball en equip.
- Desenvolupar una aplicació web completa, funcional i documentada.

---

## Com es realitza el projecte?

El projecte es desenvolupa en **5 sprints** amb durada total de **22 setmanes**:

| Sprint | Duració | Tema | Continguts principals |
|--------|---------|------|----------------------|
| Sprint 1 | 2 setmanes | Planificació i disseny | Configuració d'entorn, wireframes, Scrum |
| Sprint 2 | 3 setmanes | Estructura i maquetació | HTML semàntic, React Router, components |
| Sprint 3 | 7 setmanes | Lògica JavaScript | Variables, funcions, arrays, filtres, cerca |
| Sprint 4 | 6 setmanes | Interactivitat i DOM | Esdeveniments, formularis, localStorage |
| Sprint 5 | 4 setmanes | Comunicació asíncrona | Fetch API, JSON, mapes, presentació final |

### Metodologia

- **Scrum:** cada sprint té un producte concret lliurable.
- **Tauler Kanban (Trello):** cada equip gestiona les seues tasques amb les columnes: Backlog, Especificar, Programar, Testing i Terminada.
- **Git i GitHub:** treball amb branques per sprint (`sprint1`, `sprint2`, `sprint3`...) i commits regulars.
- **Treball en equip:** rols rotatius, revisió entre company i treball autònom.

## Tecnologies i eines

| Eina | Funció |
|------|--------|
| **React + Vite** | Framework i bundler de l'aplicació |
| **React Router** | Navegació entre vistes |
| **JavaScript ES6+** | Llenguatge de programació principal |
| **HTML5 / CSS3** | Estructura i estil |
| **Leaflet + OpenStreetMap** | Mapes interactius |
| **Fetch API** | Comunicació asíncrona amb JSON |
| **Git / GitHub** | Control de versions i col·laboració |
| **Trello** | Gestió del projecte (Kanban) |
| **VS Code** | Editor de codi |
| **Chrome / Firefox DevTools** | Depuració i proves |

## Requisits previs

1. **VS Code** amb les extensions recomanades (`Live Server`, `Prettier`, `ES7+ React/Redux/React-Native snippets`).
2. **Node.js i npm** instal·lats localment.
3. **Git** instal·lat i configurat.
4. **Compte en GitHub**.
5. **Compte en Trello** per a la gestió del projecte.
6. **Navegador web modern** (Chrome o Firefox) amb React Developer Tools instal·lat.
